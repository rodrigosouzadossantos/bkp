That happens because Plotly’s `hovertemplate` does **not render arbitrary HTML `<img>` tags** inside hover labels.

So instead of interpreting:

```html
<img src="...">
```

it escapes it as text.

Use `custom_data` + `hovertemplate` only for text, and display images another way.

## ✅ Use image URLs (or base64) in `hover_data`

OR

## ✅ Use Dash for real HTML hover cards
