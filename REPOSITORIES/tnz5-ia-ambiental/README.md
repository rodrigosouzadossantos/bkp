#  Projeto Ambiental -- Coral-Sol

## Sobre

O projeto visa a segmentação do coral-sol em imagens submarinas, além de uma relação de densidade dos corais em relação
a estrutura que está fixado.

## Organização

- "coral-sol": possui todos os códigos, notebooks e arquivos úteis em relação aos modelos de coral-sol.
- "duto": possui todos os códigos, notebooks e arquivos úteis em relação aos modelos de segmentação de dutos.
