# Coral-Sol

> Nessa pasta, estão contidos todos os arquivos referentes ao projeto ambiental, voltado a tarefa de segmentação e
> classificação de coral-sol.


> Para obter mais informações sobre o conteúdo das pastas listadas abaixo, consulte o arquivo README.md correspondente.

### Pastas principais

- automacao-ground-thuth: orquestrador de CSV e Predições - AWS Ground Thuth.
- codigo-python: arquivos referentes aos códigos rodados localmente em python, normalmente usados para tarefas mais
  genéricas, como cortar imagens, mover arquivos e plotar anotações.
- notebooks-sagemaker: Jupyter Notebooks usados no Sagemaker.
- pesos-modelos: pesos dos modelos treinados, de coral-sol.
